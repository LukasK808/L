# Setup Instructions

1. Clone the repository using `git clone https://github.com/LukasK808/L.git`
2. Navigate to the project directory: `cd L`
3. Install necessary dependencies: `npm install`

# Customization Guide

To customize the project:
- Edit the configuration files located in the `config` directory.
- Create your own custom scripts in the `scripts` directory.

For further assistance, refer to the official documentation.
